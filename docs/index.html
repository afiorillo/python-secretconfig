config()¶

Returns a list of tuples containing the current values (default or otherwise) for this configuration instance.

defaults()¶

Returns a list of tuples containing the defaults for this configuration instance.

dump(filename, overwrite=False, *args, **kwargs)¶

Writes the configuration out to the given filename.

Parameters:
  • filename – The target file to write the configuration to.
  • overwrite – (Default=False). If False and the target filename already exists, fails.
Raises:

OSError – If overwrite==False and filename already exists.
classmethod dumps(*args, **kwargs)¶

Serialized the configuration instance into the stream.

get(*args)¶

Returns the value for the given option. Can either be called get(key) for global key/value pairs or get(section, key) for section-specific key/value pairs. :raises UnknownSectionError: If the section (if given) is not available. :raises UnknownKeyError: If the given key is not available in the section (if given) or globally (if not).

getbool(*args)¶

Helper function that coerces the value to be a boolean. :raises ValueError: If the option cannot be coerced into a boolean.

getint(*args)¶

Helper function that coerces the value to be an integer. :raises ValueError: If the option cannot be coerced into an integer.

has_key(section, key)¶

Returns TF if the key is available in the given section. Returns False if either the section or key does not exist.

has_section(section)¶

Returns TF whether a SectionKV with the named section exists in this configuration instance.

items(section)¶

Returns a list of tuples in the given section. If section is None returns the global tuples.

keys(*args)¶

Returns the keys available in the given section. If section is None then returns the GLOBAL keys. :raises UnknownSectionError: If section is not in the list of available sections.

classmethod load(filenames, *args, **kwargs)¶

Loads the given filename(s) as a configuration

Attempts to read a list of potential filenames, ignoring filenames that cannot be accessed.

Parameters:
  • filenames – A filename or a list of potential filenames to try.
  • defaults – (Default=[]) A list of tuples to act as default parameters. Two-tuples map to global key/value pairs and three-tuples map to section/key/value triplets.
Raises:
  • OSError – If none of the filenames can be accessed.
  • ParsingError – If an accessible filename is incorrectly formatted (not parsable).
classmethod loads(stream, *args, **kwargs)¶

Loads a configuration from the given stream.

static parse_default(default)¶

Parses one of the elements in a defaults list. Returns either a GlobalKV or a SectionKV.

Accepts either a 2-tuple (for GlobalKV defaults) or a 3-tuple (for SectionKV defaults). 2-tuples are parsed as (key,value) and 3-tuples are parsed as (section,key,value).

sections()¶

Returns a set of available sections.

set(*args)¶

Sets the value for the given option. Can either be called set(key, value) for global key/value pairs or set(section, key, value) for section-specific key/value pairs. Overwrites any existing values.

Plaintext Parsers¶

class secretconfig.JSONConfig(defaults=None, *args, **kwargs)¶

A config parser for JSON-formatted files.

Usage:

>>> cfg = JSONConfig(defaults=[('hello', 'world'), ('foo', 'bar', 'baz')])
>>> cfg.get('hello')
'world'
>>> cfg.get('foo', 'bar')
'baz'
>>> cfg.set('foo', 'bar', 'bas')
>>> cfg.dump("test_cfg.json", overwrite=True)
>>> cfg2 = JSONConfig.load("test_cfg.json")
>>> cfg.sections()
set(['foo'])
>>> cfg.get('foo', 'bar')
'bas'
